## Usage

Add the following lines to you `build.sbt` file:

~~~ scala
// build.sbt
resolvers += Resolver.bintrayRepo("choffmeister", "maven")

libraryDependencies += "de.choffmeister" %% "auth-common" % "0.4.0"
~~~
